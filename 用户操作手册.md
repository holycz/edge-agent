## 一、插件介绍

**移点通** 是一款专为陕西移动内部同事打造的浏览器智能助手插件，基于Chrome/Edge浏览器开发。

### 主要功能

| 功能 | 说明 |
|------|------|
| 🤖 **AI 智能问答** | 针对网页内容，有问必答 |
| ✨ **文本润色** | 一键优化选中文字的表达 |
| 🔍 **内容稽核** | 检查文字中的敏感信息和错误 |
| 📄 **网页总结** | 快速总结整个网页的核心内容 |
| 📝 **批示总结** | 智能识别并汇总OA领导批示内容 |

## 二、安装与配置

### 2.1 插件安装

#### 方式一：开发者模式加载（推荐）

1. 打开 Chrome/Edge 浏览器
2. 在地址栏输入 `chrome://extensions/`（Chrome）或 `edge://extensions/`（Edge）
3. 开启右上角的「**开发者模式**」
4. 点击「**加载已解压的扩展程序**」
5. 选择项目中的 `frontend` 文件夹
6. 安装成功！🎉

#### 方式二：打包安装（IT管理员）

1. 在扩展程序页面点击「**打包扩展程序**」
2. 选择 `frontend` 文件夹
3. 生成 `.crx` 文件后分发给用户安装

### 2.2 后端服务配置

**移点通** 需要连接后端服务才能使用AI功能。

#### 查看/修改后端地址

1. 点击浏览器右上角的「**移点通**」图标
2. 在弹出窗口中查看「后端连接状态」
3. 如显示「未连接」，请检查：
   - 后端服务是否已启动
   - 网络连接是否正常
   - 后端地址配置是否正确（默认：`http://10.142.135.57:8000`）

#### 后端服务启动（IT人员）

```bash
cd backend
pip install -r requirements.txt
# 编辑 .env 文件配置 API_KEY
python run.py
```

服务默认启动在 `http://localhost:8765`

## 四、核心功能详解

### 4.1 划词智能操作 💡

在网页中选中任意文字，右键菜单即可使用AI功能。

#### 操作步骤：

1. **选中文字**：鼠标拖动选中需要处理的文本
2. **右键点击**：弹出浏览器右键菜单
3. **选择功能**：
   - **润色改写** ✨ - 优化文字表达，让内容更专业
   - **稽核检查** 🔍 - 检查敏感词、错别字、语法问题
   - **AI问答** 🤖 - 针对选中文本进行提问

#### 使用场景：

| 场景 | 推荐功能 | 示例 |
|------|----------|------|
| 写工作汇报 | 润色改写 | 优化语气和表达 |
| 审核公告文案 | 稽核检查 | 检查敏感信息和错误 |
| 理解技术文档 | AI问答 | 询问"这段代码是什么意思？" |

---

### 4.2 页面对话模式 💬

双击网页空白处，打开侧边对话面板，基于**当前网页全文**进行问答。

#### 操作步骤：

1. **打开面板**：在网页空白处**双击鼠标左键**
2. **输入问题**：在底部输入框中输入您想了解的内容
3. **查看回答**：AI 会自动结合当前页面内容回答
4. **关闭面板**：
   - 点击右上角的「×」按钮
   - 或再次双击空白处

#### 典型问题示例：

```
✅ "这篇文章的主要内容是什么？"
✅ "请提取这篇文章的关键数据"
✅ "这个方案的优点和缺点是什么？"
✅ "帮我总结一下审批流程"
```

#### 注意事项：

- 首次提问时，移点通会自动提取**当前网页的完整内容**作为上下文
- 同一对话中的后续问题，会保持上下文连贯
- 如果页面内容很长，首次加载可能需要几秒钟

---

### 4.3 网页总结 📄

一键总结整个网页的核心内容，快速把握文章要点。

#### 两种方式：

**方式一：右键菜单（未选中文本时）**
1. 在网页任意位置右键（不要选中文字）
2. 选择「移点通」→「**总结该网页**」

**方式二：图标点击**
1. 点击浏览器工具栏的「移点通」图标
2. 在弹出窗口中点击「**总结当前页面**」按钮

#### 输出内容：

- 文章核心观点
- 关键信息提取
- 内容结构梳理
- 重要数据汇总

---

### 4.4 领导批示总结 📝

专为OA审批页面设计，智能识别并汇总领导批示内容。

#### 使用场景：

- OA审批流程中查看领导批示
- 汇总多个领导的审批意见
- 快速提取与您相关的批示内容

#### 操作步骤：

1. 打开OA审批页面
2. 右键点击空白处（不选中文字）
3. 选择「移点通」→「**总结领导批示**」
4. 移点通会：
   - 识别页面中的所有批示内容
   - 标注批示人身份
   - 高亮与您相关的批示
   - 按时间顺序汇总展示

#### 个性化配置：

如需让移点通更精准地识别与您相关的批示，请联系IT管理员配置您的身份信息（姓名、职务等）。

## 五、使用技巧

### 5.1 提高回答质量的小窍门

#### ✅ 提问建议：

| 好的提问 | 效果 |
|----------|------|
| "请用表格形式对比A和B方案" | 结构化输出，便于对比 |
| "提取这篇文章中的3个关键结论" | 精准获取核心信息 |
| "用一句话总结这段内容" | 快速把握要点 |
| "这段内容的背景是什么？" | 补充上下文理解 |

#### ❌ 避免这样提问：

- 过于模糊："告诉我这个"、"分析一下"
- 过于宽泛："这篇文章讲了什么？"（长文章建议先分段）
- 跨页面提问：移点通只能基于**当前页面**内容回答

### 5.2 对话管理

- **新对话**：每次打开新页面，双击空白处会开启新的独立对话
- **历史消息**：在当前对话中，可以查看之前的问答记录
- **清空对话**：关闭并重新打开面板即可开始新对话

### 5.3 界面操作技巧

| 操作 | 说明 |
|------|------|
| 拖动面板 | 按住标题栏拖动到合适位置 |
| 调整大小 | 面板边缘会自动吸附，可尝试不同位置 |
| 复制内容 | 选中回复内容，Ctrl+C 复制 |
| 停止生成 | 回答生成过程中，点击「停止」按钮可中断 |

## 六、常见问题

### Q1: 插件安装后没有显示图标？

**可能原因及解决：**
- 浏览器没有固定扩展图标 → 点击「扩展程序」图标（拼图形状），找到「移点通」并点击「固定」
- 安装未成功 → 重新加载扩展程序
- 浏览器版本过低 → 请升级 Chrome/Edge 至最新版本

### Q2: 双击空白处打不开对话面板？

**解决方法：**
- 确认双击位置是真正的「空白处」（非按钮、图片、链接等可点击元素）
- 某些特殊网页（如PDF、视频全屏）可能不支持
- 检查是否有其他插件冲突

### Q3: 显示"后端连接失败"？

**排查步骤：**
1. 确认后端服务是否已启动
2. 检查网络连接是否正常
3. 确认后端地址配置正确（联系IT管理员）
4. 检查防火墙设置是否允许访问后端端口

### Q4: AI回答很慢或无响应？

**可能原因：**
- 页面内容过长 → 首次分析需要时间，请耐心等待
- 网络延迟 → 检查网络连接
- 后端服务繁忙 → 稍后重试
- 问题过于复杂 → 尝试简化问题或分段提问

### Q5: 润色/稽核功能没有生效？

**解决方法：**
- 确认已选中文字（鼠标拖动高亮显示）
- 检查选中的内容不是图片中的文字（图片文字无法直接识别）
- 确认右键菜单中显示的是「移点通」的相关选项

### Q6: 能否同时处理多个页面的内容？

**回答**：
- 每次对话基于**当前所在页面**的内容
- 如需对比多个页面，建议分别总结后再人工对比
- 不支持同时打开多个对话窗口

### Q7: 支持哪些浏览器？

**支持浏览器：**
- Google Chrome 88+
- Microsoft Edge 88+
- 其他基于 Chromium 内核的浏览器

**不支持：**
- Firefox（未适配）
- Safari（未适配）
- Internet Explorer（不支持）
